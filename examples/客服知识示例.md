# 客服中心常见问题解答

## 账号与登录

### 如何注册账号？
您可以通过以下方式注册账号：
1. 访问官网首页，点击右上角「注册」按钮
2. 填写手机号码，获取验证码
3. 设置登录密码（8-20位，包含字母和数字）
4. 完成注册后即可登录使用

### 忘记密码怎么办？
如果您忘记了登录密码，可以通过以下步骤重置：
1. 在登录页面点击「忘记密码」
2. 输入注册时使用的手机号码
3. 输入短信验证码
4. 设置新密码并确认
5. 使用新密码重新登录

### 账号被锁定了如何解锁？
连续5次输入错误密码，账号将被临时锁定30分钟。您可以：
- 等待30分钟后自动解锁
- 通过「忘记密码」重置密码立即解锁
- 联系在线客服人工解锁

## 支付与订单

### 支持哪些支付方式？
我们目前支持以下支付方式：
- 支付宝（推荐）
- 微信支付
- 银行卡支付（支持储蓄卡和信用卡）
- 余额支付（需先充值）

### 订单支付后可以退款吗？
退款政策如下：
- 虚拟商品（如充值卡、会员）不支持退款
- 实物商品在发货前可申请全额退款
- 已发货商品需在签收后7天内申请退货退款
- 退款将在审核通过后1-3个工作日原路返回

### 如何查看订单状态？
您可以在「我的订单」页面查看所有订单状态：
- 待支付：订单已创建，等待付款
- 已支付：付款成功，等待发货
- 已发货：商品已发出，物流配送中
- 已完成：订单签收完成
- 已取消：订单已取消

## 售后服务

### 如何申请退换货？
申请退换货流程：
1. 进入「我的订单」，找到对应订单
2. 点击「申请售后」按钮
3. 选择售后类型（退款/退货/换货）
4. 填写申请原因并上传凭证图片
5. 提交申请，等待客服审核（1-2个工作日）

### 商品有质量问题怎么办？
如收到的商品存在质量问题：
1. 请在签收后7天内申请售后
2. 上传质量问题的清晰照片或视频
3. 我们将承担往返运费
4. 可选择换货或全额退款

## 会员与积分

### 如何成为会员？
会员等级及权益：
- 普通会员：注册即享，基础服务
- 黄金会员：累计消费满500元，享95折优惠
- 铂金会员：累计消费满2000元，享9折+专属客服
- 钻石会员：累计消费满5000元，享85折+优先发货+生日礼包

### 积分有什么用？
积分可用于：
- 订单抵扣（100积分=1元）
- 兑换优惠券
- 兑换限定周边商品
- 参与积分抽奖活动

积分获取方式：消费1元积1分，评价晒单额外送20积分。
